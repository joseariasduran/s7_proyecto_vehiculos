## 📌 Descripción del Proyecto
Este proyecto consiste en el desarrollo y despliegue de una aplicación web interactiva orientada al análisis de un conjunto de datos de anuncios de venta de vehículos. A través de un *dashboard* construido con Streamlit, los usuarios pueden explorar dinámicamente las características de los automóviles, comparar métricas clave y descubrir patrones en el mercado de vehículos de segunda mano.

El objetivo principal es demostrar la capacidad de transformar datos crudos en una herramienta analítica accesible, funcional y desplegada en la nube.

## 🎯 Funcionalidades de la Aplicación
La interfaz web permite a los usuarios interactuar con los datos a través de las siguientes herramientas:
1. **Histogramas Dinámicos:** Visualización accionada por botones para entender la distribución del kilometraje, los precios y los años de los vehículos.
2. **Gráficos de Dispersión (Scatter Plots):** Análisis visual interactivo para detectar correlaciones, como la depreciación del precio de un vehículo a medida que aumenta su kilometraje (odómetro).
3. **Filtros de Datos:** Casillas de verificación (*checkboxes*) que permiten al usuario segmentar la información mostrada en pantalla en tiempo real.
